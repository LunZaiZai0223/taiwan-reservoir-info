# 臺灣水庫資訊 taiwan-reservoir-info 

## About

串接政府 Open Data API 取得水庫資訊，再搭配圖表及靜態檔案顯示特定水庫的資訊。除了顯示資訊之外，UI 有實作黑暗 / 明亮模式及 i18n 切換語系（繁中及英文）。
